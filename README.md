# Unsupervised_Learning
Unlike supervised learning, which relies on a teacher (labeled data), Unsupervised Learning is the ultimate explorer. You hand the algorithm raw, unlabeled data, and it independently uncovers the hidden structures, patterns, and relationships within.

🧠 The Core Pillars
Clustering: Grouping similar data points together without being told what the groups are (e.g., K-Means sorting customers by behavior).

Dimensionality Reduction: Compressing massive, complex datasets down to their most critical features without losing vital information (e.g., PCA stripping away noise).

Association Rules: Uncovering hidden "if-then" relationships between variables (e.g., Apriori discovering that people who buy coffee also buy mugs).
